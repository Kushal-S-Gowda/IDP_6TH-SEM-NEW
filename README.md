# FloodSense Pro 🌊

An AI-powered flood prediction, emergency response, and disaster management platform built for intelligent urban flood monitoring and evacuation planning.

## 🚀 Features

- Real-time flood risk prediction
- AI/ML-based flood analysis
- Interactive flood risk maps
- Early warning system
- Resource allocation optimization
- Emergency evacuation routing
- Citizen dashboard
- Authority command center
- Twilio SMS alert integration
- Explainable AI flood prediction
- Bengaluru flood zone monitoring

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/Kushal-S-Gowda/IDP_6TH-SEM-NEW.git
cd IDP_6TH-SEM-NEW
```

### 2. Create Virtual Environment

```bash
python -m venv floodsense_env
```

### 3. Activate Environment

#### Windows

```bash
floodsense_env\Scripts\activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔐 Environment Variables

Create a `.env` file in root directory:

```env
OPENWEATHER_API_KEY=your_key
OPENROUTE_API_KEY=your_key

TWILIO_ACCOUNT_SID=your_sid
TWILIO_API_KEY_SID=your_api_sid
TWILIO_API_KEY_SECRET=your_secret
TWILIO_FROM_NUMBER=your_number
TWILIO_ALERT_TO=your_phone
```
